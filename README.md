这是一个旨在方便哈工大同学在手机上随时随地查询空教室信息进行查询地小程序

- 数据来源：从哈工大教务处爬取
- 追求：做一个界面简洁、设计美观的空教室查询小程序
- 利益相关：没有任何私人利益目的
